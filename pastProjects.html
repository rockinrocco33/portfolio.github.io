Talisman

Talisman was a 10 week project in which Paul Bluidzuis, Alex Crowley, and myself set out to digitally recreate the popular boardgame Talisman, for education purposes only. The purpose of this project was to discover first hand how useful testing can be. A requirement of this project was that 70% of our 2500+ lines of code were tested in a meaningful way.
